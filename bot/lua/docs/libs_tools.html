Libraries and tools for Lua

Here are my libraries and tools for Lua. They are simple to install and use, but read detailed instructions anyway. Check their release data (date, size, and checksum) to make sure you have the latest version for your version of Lua. Some packages are also available as Lua rocks. All packages available here are in the public domain. Please drop me a line if you use one of my packages. All feedback is welcome. Enjoy.
