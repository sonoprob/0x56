string.find(s, pattern [, init [, plain]])

index of start of match

By returning the index of the found match this gives you a great way to carve up string.

string.match(s, pattern [, init])

capture

Returns the capture (instance of the found pattern), or nil if none were found. Quick way of getting a small string out of a big one, or of detecting whether the pattern exists.

string.gmatch(s, pattern)

iterator

The iterator repeatedly finds the next instance of the found pattern (capture), so it can be used in a generic while loop.
